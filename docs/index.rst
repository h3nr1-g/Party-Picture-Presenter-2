Party Picture Presenter 2
=========================

Party Picture Presenter 2 (PPP2) is a web based presentation software for party pictures. 
An unique feature of this software is the interaction and data exchange with presentation software via common
messenger applications like Telegram (Telegram_). This documentation shows you how to use and configure PPP2. 

.. _Telegram: https://telegram.org/
